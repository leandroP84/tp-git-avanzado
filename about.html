Descripción

Este proyecto consiste en una aplicación web simple desarrollada únicamente con HTML, con el fin de poner en práctica herramientas y procesos reales de trabajo colaborativo.

Temas aplicados

  1. Creación y uso de ramas por integrante.
  2. Resolución de conflictos de merge.
  3. Implementación de un workflow de GitHub Actions.
  4. Uso de archivos YAML para definir automatizaciones.
  5. Creación de tags para versionado del proyecto.

Importancia del trabajo

Este trabajo permite comprender cómo se organiza un equipo de desarrollo, cómo se automatizan validaciones y cómo se administran versiones de forma profesional.
