Bienvenidos

Este sitio web fue desarrollado como parte del Trabajo Práctico Integrador de la materia Git Avanzado de la Tecnicatura Universitaria en Programación.

El objetivo principal es aplicar conceptos de trabajo colaborativo con Git, resolución de conflictos, uso de GitHub Actions, archivos YAML y Git Tags.

Objetivos del proyecto

  • Trabajar con ramas en Git.
  • Simular un conflicto de merge y resolverlo correctamente.
  • Automatizar una validación con GitHub Actions.
  • Marcar versiones del proyecto utilizando tags.

Tecnologías utilizadas

  • HTML5
  • Git
  • GitHub
  • GitHub Actions
  • YAML
